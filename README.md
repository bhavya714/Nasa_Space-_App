## üöÄ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/nasa-bioscience-explorer.git

# Navigate to the project directory
cd nasa-bioscience-explorer

# Install dependencies
```
